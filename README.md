# Салон эпиляции Елены Гончаровой

Современный адаптивный сайт на чистом HTML5, CSS3 и JavaScript (ES6+).

## Структура проекта

- `index.html` — главная страница
- `css/style.css` — стили (Mobile First, до 4K)
- `js/main.js` — бургер-меню, плавная прокрутка, lazy loading
- `images/` — фото мастеров (elena.jpg, lyudmila.jpg, evgeniya.jpg, oksana.jpg, aleksandra.jpg)

## Запуск

Откройте `index.html` в браузере или используйте локальный сервер.

## Карта

На странице подключена Яндекс.Карта с приблизительными координатами Долгопрудного. Для точной метки по адресу «Лизачевское шоссе, д. 1, корп. 4» создайте свою карту в [Конструкторе Яндекс.Карт](https://yandex.ru/map-constructor/) и замените `src` у iframe в блоке «Контакты».

## SEO

В `<head>` заданы meta description и keywords. При публикации добавьте Open Graph-теги и favicon.
