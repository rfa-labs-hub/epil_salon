Как получить ключ доступа GitHub (показать инструкцию)
  1. Откройте страницу создания ключа (войдите в свой аккаунт GitHub).
  2. Нажмите кнопку «Generate new token».
  3. В поле Token name впишите: admin sugarepil.
  4. В Expiration выберите срок (например, «No expiration» или «1 year»).
  5. В Repository access выберите Only select repositories и в выпадающем списке найдите rfa-labs-hub/epil_salon.
  6. В Repository permissions найдите строку Contents и поставьте Read and write.
  7. Прокрутите вниз и нажмите Generate token.
  8. Скопируйте получившийся ключ (он начинается на github_pat_…) и вставьте в поле ниже.

Ключ сохранится только в этом браузере и нигде больше. Никто, кроме вас, его не увидит.
